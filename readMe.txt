The ONLINE QUIZ is a web application for to take online test in an 
efficient manner and no time wasting for checking the paper. The main objective
of ONLINE QUIZ is to efficiently evaluate the candidate thoroughly through a fully 
automated system that not only saves lot of time but also gives fast results. 



This project PHP  Online Quiz System Project with source Quiz System Project has
been developed in PHP, MySQL, Bootstrap, query, and JavaScript.
